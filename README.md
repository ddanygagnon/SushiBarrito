## But
Réaliser un site web **statique** d’apparence soignée, moderne et actuelleen utilisant HTML5,CSS3, et Bootstrap(version 4)

## Sujet
Les enfants d’un enseignant du CÉGEP désirent ouvrir leur restaurant de Sushi. N’ayant que 10, 8, 6et 3ans, ils n’ont pas le budget pour faire développer leur site par une grande firme.

## Architecture
Le site en question est un «One pager». C’est à dire un site ne comportant qu’une seule page. Il y aura cependant un menu qui devra rediriger vers les différentes sections de la page.  On ajoutera doncune fonctionnalité de type Bootstrap Scrollspy pour afficher la section active dans le menu.
